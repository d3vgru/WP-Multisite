Install the plugin, tell it where is your logo, and it will be in your login page. 

Based in <a href="http://wordpress.org/extend/plugins/login-logo/">Logo Login by Mark Jaquith</a>.

== Description ==

This plugin is intended to allow you to change your login logo easily. You just need to upload it to any place
and tell the system where it is. 
It differs from other plugins in its options and weight. It is intended to be lite and only change the logo. 

Try to adjust your image to 310x51px.


== Installation ==

1. Upload `custom-login-logo` folder to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. [Optional] Upload your logo image to Media Catalog and copy the URL
4. Go to Options -> Custom Login Logo
5. Paste the URL of your custom logo.
